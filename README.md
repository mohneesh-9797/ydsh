# ydsh
yet another dsh

currently, under heavy development.

## Build Requirement

* cmake 2.8+
* make
* g++ (need c++11 support)
* re2c 0.13.7.3+ (need UTF-8 support)
* libedit (need unicode support. see http://thrysoee.dk/editline/)
